initialize(webService, gameServer)

The initialize method in the Socket module is similar any other module in the pack. It is to set up data and methods needed during runtime. In this instance a new socket is created to manage websocket messages. For each connection to the server a series of events are defined to manage the menu, game server rooms, and mics statistics.
